# TESTBACKEND

## Описание
Обычный тестовый бэкенд проект на **FastAPI**, который работает
с базой данных на основе **MySQL**. Запускается на `0.0.0.0:8000` и
можно работать через **Swagger** в `/docs`


## Админ-Панель
Админ-панель реализована очень просто и без регистрации, но
с данными работать можно.
Работает на Starlette-Admin через `/admin`

## Запросы
Запросы написаны в отдельных файлах имеющие соответствующие названия
с эндпоинтами (запросами)
### Auth
* Регистрация `POST auth/register`
* Логин `POST auth/login`
* Обновление токена `POST auth/refresh`
### Tasks
* Список задач по фильтрации `GET /tasks`
* Создание задачи (только **админы**) `POST /tasks`
* Детали задачи `GET tasks/{id}`
* Обновление задачи (только **админы**: **суперадмин** меняет *любую* задачу, а **админ** - *свою*) `PUT tasks/{id}`
* Удаление задачи (только **админы**: **суперадмин** удаляет *любую* задачу, а **админ** - *свою*)  `DELETE tasks/{id}`
* Добавление комментариев под задачей `POST tasks/{id}/comments`
### Users
* Свой профиль `GET users/me`
* Обновление пользователя (изменение всех данных пользователей, кроме паролей может только сам суперадмин) `PUT users/{id}`
